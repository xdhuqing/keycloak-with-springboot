# keycloak-with-springboot

keycloak+springboot+thymeleaf: a keyloak learnning project

## 介绍
本项目主要为了学习keycloak创建。使用的组件：springboot  keycloak thymeleaf

采用standalone模式，windows下部署。
配置项：client user role realm
